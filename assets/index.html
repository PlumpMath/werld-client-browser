Werld Online

An MMORPG that runs on your browser. Well, not yet, but it will. Mostly inspired by Ultima Online and NetHack.

Screenshots

Disclaimer

I started developing Werld Online as an excuse to learn new technologies and to improve on things I've worked with before. The ultimate goal is to have an MMORPG running on your browser. To get there I'm prototyping a TTY client to drive the server design, because I don't have experience developing complex games (specially if it doesn't run in a terminal).

Currently

There isn't much done yet. For now you can use the TTY client to log in to the server using an account and password that won't even get saved in a database. After logging in you'll be able to walk around and chat with whoever is also logged in. That's it. Also, don't be surprised if you bump into a bug or three.

Run It

$ git clone git://github.com/mpereira/werld-client-tty.git

$ cd werld-client-tty

$ make

$ sudo make install

$ werld

Play It

Use ←↓↑→ or hjkl to move around. Press the enter key to be solicited for text input. Pressing enter again will send whatever you typed for everyone to see.
